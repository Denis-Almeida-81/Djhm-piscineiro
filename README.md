# DJHM Piscineiro
<br>
<br>
Site desenvolvido para divulgação de serviços de limpeza e manutenção de piscinas, com simulador de orçamento integrado.
<br>
<br>
🔗 Acesse o site: https://denis-almeida-81.github.io/Djhm-piscineiro/
<br>
<br>
## Funcionalidades

- Menu responsivo (mobile)
- Simulador de orçamento em JavaScript
- Layout responsivo
- Links para contato (WhatsApp e Instagram)
  <br>
  <br>
  ## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
<br>
<br>
